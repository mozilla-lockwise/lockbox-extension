This is just one component of the Lockbox product. Please see the Lockbox website for more documentation and context.

The Lockbox extension is a simple, stand-alone password manager that works with Firefox for desktop. It’s the first of several planned experiments designed to help us test and improve password management and online security.

Install it and sign in with your Firefox Account to take full advantage of cloud backup and next-generation encryption. Then share feedback here.

Install Lockbox

  1. Install Lockbox, and it will automatically disable Firefox’s password manager.
  2. Create an entry with a website name, URL, username, and password.
  3. Search or browse in the toolbar menu or on the full tab to find the password you need.
  4. Sign up or sign in with a Firefox Account to encrypt and your entries and back them up.
